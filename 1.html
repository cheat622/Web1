Study review

Tag

Tag: 특정글자를 강조하고 싶을 때 사용 "<~>강조하고 싶은 글자" 와 같이 사용 tag는 중첩 사용이 가능함.

강조

strong(진하게), u(밑줄), head(제목 표시)

줄바꿈

br: 단순한 줄바꿈
p: 단락 설정. CSS와 결합하여 다채롭게 이용 가능

목차

li: 목차 지정
ul, ol: 목차의 종류 지정, li의 부모 tag로 더 바깥쪽에 써야 함.
ul은 unordered list, ol은 ordered list를 의미

이미지 삽입

img: 이미지를 삽입 하는 태그
src: 이미지 경로를 지정해 주는 속성으로, 이미지 태그 사용시 반드시 지정 해야 한다.
height와 width 속성으로 크기 조정이 가능하다.

head

head: 웹페이지의 내용, 본문을 설명하는 코드들을 모아 지칭하는 태그
title: 웹페이지의 제목을 설정한다.
meta: 해당 문서의 정보인 metadata를 설명하기 위해 사용
[Meta charset="UTF-8"] 이런 식으로 사용

body

body: head의 반대로 본문 코드를 모아 지칭하는 태그

html

head와 body를 통합해서 지칭하는 태그
이 문서가 html로 작성 되었다는 것을 알리기 위해 !doctype html 이라는 태그도 달아줌

하이퍼 링크

a: anchor의 줄임말로 하이퍼링크를 걸어주는 태그
herf: 클릭시 이동할 링크를 알려주는 속성
target: 링크를 여는 방법
_self: 현재 페이지에서 열기
